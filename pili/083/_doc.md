# 霹靂朝靈闕


### 詩序
 `  六崇越、八無暇，正氣妖氛，俠邪難容  破元荒，滅天綱，聖心魔腸，誰能竟功  悔厲吝，奪予捨，三角宿命，一念獄堂  少陽昇，朝靈闕，論誰英雄，還看神州`

### 簡介
>  猂劫兵烽，無暇暗劫起，猂族利用地獄鳥力量，動盪苦境之後，雌伏多年的暗劫八無暇，也趁勢再出，為阻止這股邪惡勢力顛覆中原武林，湯問夢澤山座白象．香六牙，廣發天下舉鋒帖，召開擎天會，聚集西窗月、墨如鴉、不全鷹等湯問英傑，正心六崇越宗主巍翼、太叔望，以及儒道釋代表相繼響應，血漫河山俠骨烈，亙古之爭，勝敗由天抑由人？三峰深霧中，未孚不知處，墨殤天爆弭平之後，三教頂峰也從此音信杳然，絕跡江湖，泥菩薩金口斷言三人身困未孚之中，唯有雲中陌客能將他們救回。少陽君超凡入世，身負三教頂峰的驚世修為的他，以尋回失蹤的三人為己任，司馬駿業心繫摯友，為此針對少陽君，來歷成謎的他，對過去卻是諱莫如深，獨闖世道俠腸熱，一念之差，正邪由人抑由己？遊師化天下，萬願一同歸，神秘高人遊師無方，冷峻若冰山，端坐水中央，御女聖道行，裁斷蒼生望，他將率領神秘的教派萬願同歸，開創怎樣的未來？隱士林中，不朽的劍魔，魔師傲神州，暌違多年，再現塵寰，因與兵災雪三劍交易，引動俠、邪局勢擺盪，仍是紛亂不休的江湖，依舊人心叵測的世道，熟悉的中原武林，難了的情仇糾葛，盡看三條路的抉擇結果，欲知精彩劇情，請勿錯過２０２１年霹靂國際多媒體最新力作──霹靂朝靈闕。
