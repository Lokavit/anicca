# 霹靂天機


### 詩序
 `  天機啟，萬象大衍造玄奇  血路承，穹宇浩劫自太曦  氣數轉，璇璣一撥誰鬥棋  風雲合，且任逍遙共忘機`

### 簡介
>  霹靂天機係全新系列作，由天跡、地冥雙主角領銜主演，雙主線開啟新故事。看天跡如何在逆勢之中，統領群俠，對抗鯨吞中原的太曦神照。而失聯久時的地冥，悄悄集結數名能人暗中行動，他為何不現身，又將翻攪何等風雲？霹靂天機有新一代的偶像陣容，武林群星熠熠爭輝。看一劍忘機道軒眉飄然入世，將為未來武林，扭轉何等乾坤？看暗潮冷嘯伐樓那踏入苦境，他要如何尋得聖焰之源，解救家國之變？更看白堊曠野的占星師――星橋引者解璇璣，如何遊走正邪之間，卜測天下之謎。霹靂天機有堅強的反派主力，看掌握巨大寰界能量的太曦神照，併吞魔羅血界，廣納各方能人，如何獨寫霸業？看陰陽裂界父子梟雄――帝裔八龍天、往世閻摩如何動亂人世？也看八龍天、伐樓那「父殺子」預言，峰迴路轉如何演變？霹靂天機以嶄新的編排方式，講述霹靂武林新一頁的故事。第一部共分四大篇章，首篇【裂界】篇以雙線並進的方式，帶出過往兩年，天跡流落裂族，如何抽絲剝繭，解開一場「尚未看見題目的考驗」；另一邊，則帶出苦境的渾沌現況。接續而來的【魔熵】、【西崦】等篇，更將串聯玄黃三乘、靖玄九星，至六蝕玄曜以來，橫跨多檔，眾多未解之謎，更多牽引未來武林風雲變化之秘，盡在全新的霹靂天機。萬眾期盼，連臺好戲，２０２３年，千萬不可錯失，霹靂國際多媒體年度鉅獻，黃文擇布袋戲最新強檔力作――霹靂天機。
